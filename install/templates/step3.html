所需配置 当前服务器
操作系统 不限
PHP 版本 5.0.0+
附件上传 建议10M+ 允许/最大上传:
MySQL 数据库 3.23+ 支持MYSQL数据库未安装
SESSION 必须支持 已支持SESSION传值故障，请保持php.ini中的session.save_path目录可写
GD模块 必须开启 已开启GD模块未安装，请将php.ini中的;extension=php_pdo.dll的;去掉
PDO模块 必须开启 已开启PDO模块未安装，请在php.ini中加入extension=php_pdo.dll
PDO_MySQL模块 必须开启 已开启PDO_MYSQL模块未安装，请在php.ini中加入extension=php_pdo_mysql.dll
Allow_url_fopen 需要开启 已开启allow_url_fopen未开启，请到php.ini设置allow_url_fopen = On
服务器DNS解析 需要开启 已开启服务器DNS解析，没有开启(Linux 在/etc/reslov.conf设置正确的DNS)
注意:网站生成静态，必须保证 Allow_url_fopen 和 服务器DNS解析 同时开启
以上两选项，如果没有开启，网站仅不能生成静态页面，但网站仍可正常使用
